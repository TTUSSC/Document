# 社團財產管理辦法

## 社團報帳辦法

請各位在以==社團名義==買東西時，請記得==報統編跟留存發票==，後盡快繳交給社團總務。總務會記錄在當月的負債表內(會放在社團的雲端供大家檢閱)格式如附表1，並在隔月的==月初==一併將錢繳還，已繳清的帳會在負債表內標記。
